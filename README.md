# COE3200: Introduction to Tools Used for Non-Verbal Communication 

## Project Overview

This repository is part of the **COE3200-Communication Skills for Computing** course. The primary objective is to demonstrate how GitHub functions as a **non-verbal communication tool** in the IT industry. By maintaining this repository, I am showcasing my ability to use version control to present coding skills, projects, and professional standards to others.

## Repository Purpose

The purpose of this repository is to fulfill the requirements for **Part 1: GitHub Profile Creation**. It serves as a practical exercise in setting up a professional online presence and sharing technical work through a public platform.
